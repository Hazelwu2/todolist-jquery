## 🛖 About This Project
含有 index.html、index.css、index.js，用 scss 切版配合 Prepros 快速轉換 css，RWD 並加上 jQuery 實作
部署採用 Netlify 

## 📖 功能
* Create：建立 Todo
* Delete：刪除 Todo
* 篩選：全部、待完成、已完成 Todo
* 響應式設計

## 🔨 Built With
